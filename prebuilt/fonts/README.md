# Circle OS — Bundled Fonts

Place font TTF files here before building. They are referenced by
`overlay/CircleTheme/res/font/comfortaa.xml`.

## Required files

| File | Weight | Licence |
|------|--------|---------|
| `comfortaa_regular.ttf` | 400 | OFL — Google Fonts |
| `comfortaa_medium.ttf`  | 500 | OFL — Google Fonts |
| `comfortaa_bold.ttf`    | 700 | OFL — Google Fonts |

Download from: https://fonts.google.com/specimen/Comfortaa

Body text (Arial) uses the system Arial or falls back to Roboto — no TTF needed.
